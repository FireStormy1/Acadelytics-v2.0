# 🎓 Acadelytics V2

A modern academic analytics platform designed for students to calculate, track, and visualize their academic performance with ease. Acadelytics V2 provides SGPA & CGPA calculation, semester-wise progress tracking, performance insights, and a clean, responsive user experience.

# 📌 About the Project

Acadelytics V2 is the next-generation version of Acadelytics, built to simplify academic performance tracking for university students.

Managing grades across multiple semesters can become complicated and time-consuming. Acadelytics V2 eliminates manual calculations and spreadsheets by providing a centralized platform for academic analysis.

Students can calculate semester performance, monitor cumulative progress, compare results across semesters, and gain valuable insights into their academic journey through a clean and intuitive interface.

The platform is designed with a modern UI inspired by professional productivity tools, emphasizing speed, accessibility, and simplicity.

# 🧮 Formula Used

## SGPA Formula

SGPA is calculated using:

**SGPA = Σ(Credit × Grade Point) ÷ Σ(Credits)**

---

## CGPA Formula

CGPA is calculated using:

**CGPA = Σ(SGPA × Semester Credits) ÷ Σ(All Credits)**

# ⚡ How It Works

1. Select your academic branch.
2. Choose the semester.
3. Enter grades or marks for each subject.
4. Calculate SGPA instantly.
5. Save semester performance.
6. Add multiple semesters.
7. Generate overall CGPA.
8. Analyze academic progress through the dashboard.
